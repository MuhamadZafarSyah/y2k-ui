## Installation

<CodeBlock language="bash" filename="terminal.sh" code="npx y2kui@latest add breadcrumb" />

## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbEllipsis,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@/components/ui/breadcrumb"
```

## Parts

| Part | Description |
|------|-------------|
| `Breadcrumb` | Nav wrapper with aria-label |
| `BreadcrumbList` | OL with thick border and card bg |
| `BreadcrumbItem` | List item wrapper |
| `BreadcrumbLink` | Clickable link with pink hover |
| `BreadcrumbPage` | Current page (bold, no link) |
| `BreadcrumbSeparator` | ChevronRight icon separator |
| `BreadcrumbEllipsis` | Truncated "···" indicator |
