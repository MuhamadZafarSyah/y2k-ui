# Slider

A flat range slider — thick navy border, lemon thumb, five color variants.
Fully controlled / uncontrolled.

## Bare (no props)

The simplest usage: a single thumb, no label. Useful when the slider sits
inside a labeled section or when the parent provides context.

<ComponentPreview
  name="slider-bare-demo"
  preview={<SliderDemo />}
  source={`import { Slider } from "@/components/ui/slider"

<Slider value={[42]} onValueChange={() => {}} className="w-72" />`}
/>

## With props (label, value, trailing unit, range)

Pass `label`, `showValue`, and `trailingLabel` to render a labelled slider.
Use an array value for a **range slider** with two thumbs (great for min/max).

<ComponentPreview
  name="slider-with-label-demo"
  preview={<SliderWithLabelDemo />}
  source={`import { Slider } from "@/components/ui/slider"

const [volume, setVolume] = React.useState([42])

<Slider
  label="Volume"
  showValue
  trailingLabel="dB"
  value={volume}
  onValueChange={setVolume}
  min={0}
  max={100}
/>

<Slider
  label="Sweetness"
  showValue
  value={[30, 70]}
  min={0}
  max={100}
  step={5}
  variant="pink"
/>`}
/>
