# Avatar

A retro-square avatar (no rounded pills) with a thick navy border. Five
sizes, image + fallback slots, and an `AvatarGroup` overlap layout.

## Usage

```tsx

  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarGroup,
} from "@/components/ui/avatar";
```
