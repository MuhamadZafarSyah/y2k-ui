# Avatar

A retro-square avatar (no rounded pills) with a thick navy border. Use `AvatarImage` for a real image, `AvatarFallback` for initials.

<ComponentPreview
  name="avatar-demo"
  preview={<AvatarDemo />}
  source={`import { Avatar, AvatarImage, AvatarFallback } from "@/components/ui/avatar"

<Avatar>
  <AvatarImage src="/me.png" alt="@me" />
  <AvatarFallback>ME</AvatarFallback>
</Avatar>`}
/>

## Installation

<CodeBlock language="bash" filename="terminal.sh" code="npx y2kui@latest add avatar" />

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@/components/ui/avatar"
```
