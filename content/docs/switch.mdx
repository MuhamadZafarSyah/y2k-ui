# Switch

A flat toggle — pastel mint track when on, white when off, with a lemon
knob that has a thick navy border.

  name="switch-demo"

<Switch checked={on} onCheckedChange={setOn} />`}
/>

## Installation

<CodeBlock
  language="bash"
  filename="terminal.sh"
  code="npx y2kui@latest add switch"
/>

## Usage

```tsx
import { Switch } from "@/components/ui/switch";
```
