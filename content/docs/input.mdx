# Input

A flat text input. Thick navy outline, soft pink focus ring, with optional
add-ons for leading/trailing icons and prefix/suffix labels.

## Basic

<ComponentPreview
  name="input-basic-demo"
  preview={<InputBasicDemo />}
  source={`import { Input } from "@/components/ui/input"
import { MailIcon } from "lucide-react"

<div className="flex flex-col gap-3">
  <Input placeholder="Default text input" />
  <Input type="email" placeholder="you@y2k.ui" leadingIcon={<MailIcon />} />
</div>`}
/>

## With icons

Pass any React node through `leadingIcon` or `trailingIcon` — useful for
search bars, password reveals, or action shortcuts.

<ComponentPreview
  name="input-icons-demo"
  preview={<InputIconsDemo />}
  source={`import { Input } from "@/components/ui/input"
import { SearchIcon, MailIcon, EyeIcon, PlusIcon } from "lucide-react"

<Input placeholder="Search the docs…" leadingIcon={<SearchIcon />} />

<Input
  type="password"
  placeholder="Password"
  leadingIcon={<MailIcon />}
  trailingIcon={<EyeIcon />}
/>

<Input placeholder="Add a tag" trailingIcon={<PlusIcon />} />`}
/>

## With prefix & suffix

Use `prefix` for currency / symbol labels and `suffix` for units. They render
in a small pastel `panel` chip so the value stays readable.

<ComponentPreview
  name="input-prefix-suffix-demo"
  preview={<InputPrefixSuffixDemo />}
  source={`import { Input } from "@/components/ui/input"
import { DownloadIcon } from "lucide-react"

<Input
  type="number"
  prefix="Rp"
  suffix="IDR"
  placeholder="0"
  leadingIcon={<DownloadIcon />}
/>

<Input type="number" prefix="$" suffix="USD" placeholder="0" />
<Input type="number" suffix="kg" placeholder="0" />
<Input type="text" prefix="@" placeholder="username" />`}
/>

```tsx
import { Input } from "@/components/ui/input"
```
