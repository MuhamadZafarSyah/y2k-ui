# Charts

Interactive chart components powered by [Recharts](https://recharts.org) with Y2K styling — thick navy outlines, pastel fills, custom tooltips, and retro window frames.

## Installation

<CodeBlock language="bash" filename="terminal.sh" code="npx y2kui@latest add chart" />

## Usage

```tsx
import { BarChart, LineChart, DonutChart } from "@/components/ui/chart"
```
