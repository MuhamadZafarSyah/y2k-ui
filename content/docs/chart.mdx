# Charts

Interactive chart components powered by [Recharts](https://recharts.org) with Y2K styling — thick navy outlines, pastel fills, custom tooltips, and retro window frames.

## Installation

<CodeBlock
  language="bash"
  filename="terminal.sh"
  code="npx y2kui add chart"
/>

## Bar Chart

A flat bar chart with thick navy outlines, pastel fills, and a custom Y2K tooltip. Great for comparing discrete values across categories.

### Preview

<ComponentPreview
  name="bar-chart-demo"
  preview={<BarChartDemo />}
  source={`import { BarChart } from "@/components/ui/chart"

const data = [
{ label: "Mon", value: 120 },
{ label: "Tue", value: 250 },
{ label: "Wed", value: 180 },
{ label: "Thu", value: 320 },
{ label: "Fri", value: 210 },
]

<BarChart data={data} />`}
/>

### Usage

```tsx
import { BarChart } from "@/components/ui/chart";

const data = [
  { label: "Mon", value: 120 },
  { label: "Tue", value: 250 },
  { label: "Wed", value: 180 },
  { label: "Thu", value: 320 },
  { label: "Fri", value: 210 },
];

export function Example() {
  return <BarChart data={data} />;
}
```
