# Carousel

A scrollable carousel with previous/next buttons, powered by Embla.

<Carousel className="w-full max-w-xs">

          {item}

  <CarouselPrevious />
  <CarouselNext />

/>

## Installation

<CodeBlock language="bash" filename="terminal.sh" code="npx y2kui add carousel" />

## Usage

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselPrevious,
  CarouselNext,
} from "@/components/ui/carousel"

<Carousel className="w-full max-w-xs">
  <CarouselContent>
    {items.map((item, i) => (
      <CarouselItem key={i}>
        <div className="p-4">{item}</div>
      </CarouselItem>
    ))}
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```
