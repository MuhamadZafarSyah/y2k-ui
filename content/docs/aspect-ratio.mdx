# Aspect Ratio

A container that maintains a specific aspect ratio for its content.

<ComponentPreview
  name="aspect-ratio-demo"
  preview={<AspectRatioDemo />}
  source={`import { AspectRatio } from "@/components/ui/aspect-ratio"

<div className="w-72">
  <AspectRatio ratio={16 / 9}>
    <img src="..." alt="..." />
  </AspectRatio>
</div>`}
/>

## Installation

<CodeBlock language="bash" filename="terminal.sh" code="npx y2kui add aspect-ratio" />

## Usage

```tsx
import { AspectRatio } from "@/components/ui/aspect-ratio"

<AspectRatio ratio={16 / 9}>
  <img src="..." alt="..." />
</AspectRatio>
```
